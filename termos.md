### 1.2 Descrição do Serviço
O Facilita Estudante LITE é uma plataforma experimental de produtividade e gerenciamento de tarefas focada em dispositivos de baixo desempenho. O serviço é fornecido "como está" e "conforme disponível".

### 1.3 Uso de Chaves de API e Credenciais (OpenRouter / Inter IA)
* **Responsabilidade do Usuário:** O aplicativo permite a integração com serviços de inteligência artificial através de chaves de API próprias do usuário.
* **Segurança da Chave:** O usuário é o único responsável por manter o sigilo de suas credenciais.

### 1.4 Segurança Local (Bloqueio por PIN)
* O recurso de bloqueio por PIN (4 a 8 caracteres) serve apenas para evitar edições acidentais de tarefas na interface local.
* Esta função não substitui a segurança física ou digital do próprio sistema operacional do dispositivo do usuário.

### 1.5 Limitação de Responsabilidade
A Intervixus Corp. não garante o funcionamento ininterrupto do app e não se responsabiliza por perda de dados de tarefas, falhas de cache ou por respostas imprecisas geradas pelas IAs de terceiros (Gemma, Nemotron, LFM).

## 2. POLÍTICA DE PRIVACIDADE

### 2.1 Armazenamento Local de Dados
* **Dados de Tarefas:** Títulos, descrições, prazos, prioridades, estatísticas e o PIN de segurança são salvos estritamente no armazenamento local do seu navegador (LocalStorage/IndexedDB).
* **Armazenamento de Chaves:** As chaves de API inseridas ficam guardadas no seu dispositivo para realizar as requisições diretas ao provedor.

### 2.2 Compartilhamento com Terceiros
* Ao utilizar a função de melhoria de texto da Inter IA, os dados da respectiva tarefa são transmitidos via API para os servidores da OpenRouter e dos provedores do modelo selecionado.
* **Recomendação importante:** Nunca insira dados pessoais sensíveis, senhas, dados bancários ou informações confidenciais nos campos de tarefas enviados para a IA.

### 2.3 Cookies e Preferências
O aplicativo utiliza apenas tecnologias locais técnicas e estritamente necessárias para salvar suas preferências de acessibilidade, tais como o Modo Escuro, tamanho da fonte e tamanho do cursor.

### 2.4 Controle e Direitos do Usuário (LGPD)
Você detém o controle total sobre seus dados. Como as informações não são enviadas para um banco de dados centralizado da Intervixus Corp., você pode apagar permanentemente todas as suas tarefas, chaves de API e configurações a qualquer momento limpando o cache e os dados de navegação do seu navegador web.
